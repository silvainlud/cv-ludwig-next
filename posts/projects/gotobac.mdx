Gotobac fut mon premier projet reposant sur plusieurs technologies :
- PHP pour le site web
- Java (Android) pour une application mobile
- Java pour le web service
- Firebase pour la gestion des notifications mobiles
- Python : pour l'envoi des notifications à Firebase

L'objectif de ce site est d'offrir aux étudiants de terminale une plateforme leur permettant de gérer leurs cours. Ce
site permettait le stockage de leurs cours et également la gestion de projet comme ceux proposés en science de
l'ingénieur. La dernière version de la plateforme implémentait le principe de responsable de classe (délégué), ce
dernier pouvait alors inviter des camarades dans sa classe. Il existait également un système de chat entre deux ou
plusieurs élèves.

Ce site a été utilisé lorsque j'étais en terminale mais, en raison des coûts dû à l'infrastructure, la plateforme n'est
plus en ligne à ce jour. Je me laisse la possibilité de la relancer.
